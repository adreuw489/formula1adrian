Capítulo 1. HTML
Es el lenguaje estándar para la construcción de páginas web.
La página web contiene solamente texto. Para añadir un elemento a la página como una imagén, vídeo, script, etc. no se inscrusta directamente en el código de la página, sino que se hace una referencia a la ubicación de dicho elemento.
El navegador realiza la tarea de unir todos los elementos y visualizar la página final.
