Capítulo 3. XML
Es un meta-lenguaje que permite crear otros lenguajes de marcas y que se utiliza para almacenar datos.
La utilidad principal de XML es que se permite almacenar datos con estructuras sin necesidad de usar un SGBD, es decir, permite construir una base de datos ligera. Esto permite intercambiar informaciónn entre sistemas que de otra forma serían incompatibles.
